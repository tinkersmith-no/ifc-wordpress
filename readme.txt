Displays the current International Fixed Calendar date using a shortcode.

== Description ==
Use the [ifc_date] shortcode to display today's International Fixed Calendar date.

== Installation ==
1. Upload the plugin and activate it.
2. Add the [ifc_date] shortcode where you want the date displayed.

== Frequently Asked Questions ==
= What is the International Fixed Calendar? =
The International Fixed Calendar is an alternative to the Gregorian calendar. It consists of 13 months with 28 days each, plus one or two additional intercalary days. Each month begins on a Monday and ends on a Sunday.

More information is available at:
https://tinkersmith.no/international-fixed-calendar
